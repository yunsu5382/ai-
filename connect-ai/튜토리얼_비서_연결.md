# 비서(Secretary) 연결 튜토리얼

핸드폰 텔레그램으로 회사 비서와 자연어로 대화하고, 비서가 구글 캘린더에 직접 일정을 잡거나 빼게 만드는 셋업입니다. **둘 다 합쳐서 10분이면 끝.**

---

## 1️⃣ 텔레그램 봇 연결 (5분)

### Step 1. 봇 만들기
1. 핸드폰 텔레그램에서 **[@BotFather](https://t.me/BotFather)** 검색
2. `/newbot` 입력
3. 봇 이름 입력 (예: `우리회사 비서`)
4. 봇 username 입력 — 끝이 `bot`이어야 함 (예: `myco_secretary_bot`)
5. **토큰 받음** — `123456789:AAEx...` 형태. 메모해두기

### Step 2. 봇과 대화 시작
1. BotFather가 알려준 봇 링크 클릭 (또는 username 검색)
2. **`Start`** 버튼 누르기 (아무 메시지나 한 번 보내야 다음 단계에서 chat_id가 잡힘)

### Step 3. VS Code/Cursor에서 연결
1. 사이드바에서 **Connect AI** 아이콘 클릭 → **비서(Secretary)** 선택
2. **"📨 텔레그램이 아직 연결 안 됐어요"** 카드 클릭 (또는 Skills → ⚙️)
3. 3단계 마법사:
   - **봇 토큰**: 위에서 받은 토큰 붙여넣기 → 자동 검증 ✅
   - **Chat ID**: 봇과 대화 시작한 계정이 자동 감지됨 → 클릭
   - **테스트 메시지 발송** → 핸드폰에 ✅ 도착하면 성공
4. 끝!

### 확인
핸드폰 봇에 **"안녕"** 보내기 → 비서가 답장하면 완료.

---

## 2️⃣ 구글 캘린더 연결 (5분)

비서가 일정을 **추가/조회/취소** 다 하려면 OAuth 연결이 필요합니다.

### Step 1. OAuth 연결 시작
1. VS Code/Cursor에서 `Cmd+Shift+P` (맥) / `Ctrl+Shift+P` (윈도우)
2. **"Connect AI: Google Calendar 자동 일정 연결 📅"** 입력 후 엔터

### Step 2. 구글 인증
1. 브라우저가 자동으로 열림
2. 본인 구글 계정 선택
3. "Connect AI"가 캘린더 접근 권한 요청 → **허용**
4. "인증 완료" 페이지 뜨면 브라우저 닫고 IDE로 돌아오기

### Step 3. 확인
사이드바 채팅창 또는 텔레그램에서 "**오늘 일정 뭐야?**" 입력 → 캘린더 일정이 떠오면 성공.

---

## 3️⃣ 비서와 대화하기

이제 핸드폰 텔레그램으로 회사를 굴리세요. **외울 명령 없음, 그냥 자연어로.**

### 일정 관리
| 이렇게 말하면 | 비서가 하는 일 |
|---|---|
| "오늘 일정 뭐야?" | 캘린더 조회해서 알려줌 |
| "내일 3시 광고주 미팅 잡아줘" | 캘린더에 1시간짜리 일정 자동 생성 |
| "내일 미팅 취소해" | 일정 찾아서 삭제 |
| "이번 주 비는 시간 알려줘" | 빈 슬롯 분석 |

### 작업 분배
| 이렇게 말하면 | 비서가 하는 일 |
|---|---|
| "유튜브 트렌드 분석해줘" | CEO에게 전달 → 적합한 에이전트가 결과 보고 |
| "썸네일 시안 3개 만들어" | 디자이너에게 분배 |
| "포트폴리오 사이트 만들어" | 개발자에게 분배 |

### 상태 확인
| 이렇게 말하면 | 비서가 하는 일 |
|---|---|
| "할일 뭐 있어?" | 추적 중인 작업 목록 |
| "에이전트 뭐 하고 있어?" | 진행 중 작업 + 최근 활동 보고 |
| "어제 뭐 했어?" | 의사결정 로그 + 세션 보고 요약 |
| "도구 자율도 어때?" | 각 에이전트 권한 레벨 |

### 작업 완료/취소 (안전장치)
모호함 방지를 위해 ID 기반:
- `/done abc123` — 작업 완료 처리
- `/cancel abc123` — 작업 취소

ID는 "할일 뭐 있어?"라고 물어보면 비서가 알려줌. 마지막 9자리만 입력해도 OK.

## 🆘 안 될 때

**텔레그램 답이 안 와요**
- 봇 토큰이 정확한지 확인 (`/start`로 BotFather 다시 → `My Bots` → 토큰 확인)
- 봇과 한 번이라도 대화한 적 있는지 (Step 2 빠뜨림)

**같은 답이 여러 번 와요**
- VS Code/Cursor 창이 여러 개 떠있으면 각자 답해서 그래요
- 회사 폴더 띄운 창 1개만 남기고 나머지 닫기

**캘린더 연결이 풀려요**
- OAuth 토큰이 만료된 것 — 명령 팔레트에서 "Google Calendar 자동 일정 연결" 다시 실행

**비서가 "다시 한 번 말씀해주실 수 있나요?"라고만 해요**
- 모델이 답을 끝내지 못한 것 — 한 번 더 보내면 됨
- 자주 발생하면 LM Studio/Ollama 모델을 더 큰 걸로 교체

## 핵심
- **외울 명령은 사실상 없음** — 자연어로 그냥 말하기
- **비서는 24시간 대기** — 자리 비워도 핸드폰으로 회사가 뭐 하는지 봄
- **에이전트 작업 결과**는 자동으로 텔레그램에 요약돼서 도착
- **민감 정보**(봇 토큰, OAuth 토큰)는 `_agents/secretary/tools/`에 로컬 저장, git 자동 제외됨
